1. Определение и свойства

Правильная пирамида — это пирамида, у которой:

  1. Основание — правильный многоугольник (квадрат, треугольник и т.д.)
  2. Боковые рёбра равны между собой
  3. Высота проходит через центр основания

3D модель правильной пирамиды

Ключевые свойства:

  • Все боковые грани — равные равнобедренные треугольники
  • Апофема (высота боковой грани) одинакова для всех граней
  • Углы наклона боковых рёбер к плоскости основания равны
  • Центр основания совпадает с проекцией вершины

2. Основные элементы

Элемент Обозначение Описание
Основание ABCD Правильный n-угольник
Вершина S Точка, не лежащая в плоскости основания
Боковое ребро SA, SB, ... Отрезок, соединяющий вершину с вершиной основания
Высота SO Перпендикуляр из вершины к основанию
Апофема SH Высота боковой грани

Пример обозначений для четырёхугольной пирамиды:

SABCD - пирамида
ABCD - квадратное основание
SO = h - высота
SH = l - апофема

Примечание: В правильной пирамиде все боковые грани равны, поэтому апофема одинакова для всех граней.

3. Формулы и вычисления

3.1. Для правильной n-угольной пирамиды

Площадь боковой поверхности:

Sбок = ½ × P × l

где P — периметр основания, l — апофема

Апофема (l) — это высота боковой грани, проведённая из вершины пирамиды к стороне основания.

Площадь полной поверхности:

Sполн = Sбок + Sосн

где Sосн — площадь основания

Объём:

V = ⅓ × Sосн × h

где h — высота пирамиды

3.2. Для правильной четырёхугольной пирамиды

Связь апофемы и высоты:

l = √(h² + (a/2)²)

где a — сторона квадратного основания

Подсказка: как вывести эту формулу? Формула выводится из прямоугольного треугольника, образованного высотой пирамиды (h), половиной стороны основания (a/2) и апофемой (l) как гипотенузой.

Вывод формулы апофемы:

1. Рассмотрим прямоугольный треугольник SOH (O - центр основания, H - середина стороны AB)

2. По теореме Пифагора: SH² = SO² + OH² ⇒ l² = h² + (a/2)²

На рисунке показано, как апофема (l), высота пирамиды (h) и половина стороны основания (a/2) образуют прямоугольный треугольник.

4. Типовые задачи

4.1. Нахождение элементов по известным параметрам

Задача 1: Найдите высоту правильной четырёхугольной пирамиды, если сторона основания равна 6 см, а боковое ребро 5 см.

Решение:

1. Находим диагональ основания: d = a√2 = 6√2 см

2. Половина диагонали: AO = d/2 = 3√2 см

3. Из ΔSAO по теореме Пифагора: h = √(SA² - AO²) = √(25 - 18) = √7 см

Ответ: √7 см ≈ 2.645 см

4.2. Задачи на сечения

Задача 2: В правильной треугольной пирамиде со стороной основания 4 см проведите сечение через середину высоты параллельно основанию. Найдите площадь сечения.

Решение:

1. Сечение — подобный основанию треугольник с коэффициентом подобия ½

2. Площадь основания: Sосн = (a²√3)/4 = (16√3)/4 = 4√3 см²

3. Площадь сечения: S = (½)² × 4√3 = √3 см²

Ответ: √3 см² ≈ 1.732 см²

5. Примеры решений

Пример 1. Полная поверхность пирамиды

В правильной четырёхугольной пирамиде сторона основания равна 8 см, а высота — 3 см. Найдите площадь полной поверхности.

Решение:

1. Площадь основания: Sосн = a² = 8² = 64 см²

2. Находим апофему: l = √(h² + (a/2)²) = √(3² + 4²) = √(9 + 16) = 5 см

3. Площадь боковой поверхности: Sбок = ½ × P × l = ½ × (4×8) × 5 = ½ × 32 × 5 = 80 см²

4. Полная площадь: S = Sосн + Sбок = 64 + 80 = 144 см²

Ответ: 144 см²

Пример 2. Объём пирамиды

Найдите объём правильной треугольной пирамиды, если сторона основания 6 см, а боковое ребро 5 см.

Решение:

1. Площадь основания: S = (a²√3)/4 = (6²√3)/4 = 9√3 см²

2. Радиус описанной окружности: R = (a√3)/3 = (6√3)/3 = 2√3 см

3. Высота: h = √(5² - (2√3)²) = √(25 - 12) = √13 см

4. Объём: V = ⅓ × 9√3 × √13 = 3√39 см³

Ответ: 3√39 см³ ≈ 18.73 см³

6. Применение в реальном мире

Пирамидальные конструкции широко используются в архитектуре и инженерии благодаря своей устойчивости и оптимальному распределению нагрузки.

  • Архитектура: Пирамиды Гизы, Лувр в Париже, современные небоскрёбы
  • Инженерия: Антенные конструкции, опоры ЛЭП
  • Природа: Кристаллические решётки минералов, форма молекулы аммиака

Практическое значение формул:

Расчёт материалов для строительства, определение устойчивости конструкций, проектирование освещения (углы падения света на грани).
